Linux下某个文件夹有多个txt文件，目的是将txt文件里的文本格式化，要求如下：
1、
1、删除空白行
2、删除每行的开头的空白字符
3、查找以“第XXX章”开头的行文本,(XXX代表中文数字或者英文数字)，转换为“# 第XXX章”格式
4、每行增加2个全角空格
5、将以单引号‘XXX’格式文本，转换为双引号“XXX”格式
6、将以「XXX」格式的文本，转换为双引号“XXX”格式
7、每一行后增加一个空白行



#!/bin/bash

# 遍历文件夹中的所有txt文件
for file in /path/to/folder/*.txt; do
    # 1. 删除空白行
    # 2. 删除每行开头的空白字符
    # 3. 将以“第XXX章”开头的行转换为“# 第XXX章”
    # 4. 每行增加2个全角空格
    # 5. 将单引号‘XXX’转换为双引号“XXX”
    # 6. 将「XXX」转换为双引号“XXX”
    # 7. 每一行后增加一个空白行
    sed -e '/^$/d' \
        -e 's/^[[:space:]]*//' \
        -e 's/^第\([一二三四五六七八九十0-9]\+\)章/# 第\1章/' \
        -e 's/^\(.*\)/　　\1/' \
        -e "s/‘\([^’]*\)’/“\1”/g" \
        -e "s/「\([^」]*\)」/“\1”/g" \
        "$file" | awk '{print $0 "\n"}' > "${file}.tmp"
    
    # 将处理后的文件覆盖原文件
    mv "${file}.tmp" "$file"
done

根据中国现行的军工保密资格认定相关规定，申请一级、二级、三级保密资格的单位在审查程序上存在差异，并非所有级别都必须由国家军工保密资格认定委员会进行现场审查。具体规定如下：

一级保密资格
由国家军工保密资格认定委员会（简称“国家认定委”）负责组织现场审查，审查通过后由该委员会审批。

二级保密资格
通常由省级军工保密资格认定委员会（简称“省认定委”）组织现场审查，但部分特殊单位或跨省业务单位可能由国家认定委直接审查。审查通过后报国家认定委备案。

三级保密资格
一般由省级认定委组织现场审查并审批，报国家认定委备案。国家认定委通常不直接参与三级资格的现场审查。

根据中国现行的保密资格管理规定，“凡是雇佣外籍人员的单位都不能申请保密资格”这一说法不完全准确。具体情况如下：

1. 一般情况下限制雇佣外籍人员
申请保密资格的单位通常要求无外商（含港澳台）投资和雇用外籍人员，这是出于国家安全考虑，防止境外势力获取国家秘密24。
涉密岗位的工作人员必须具有中国国籍，且不得与境外人员（含港澳台）有婚姻关系24。
2. 特殊情况下允许雇佣外籍人员
经国家或省级主管部门批准的单位，即使有外商投资或雇佣外籍人员，仍然可以申请保密资格47。
外籍人员不得接触机密级以上文件，其权限需在系统中单独隔离1。
如果外籍人员因工作需要知悉国家秘密，必须经批准并签订保密协议，但不直接认定为涉密人员9。
3. 违规雇佣外籍人员的后果
已取得保密资格的单位擅自雇佣外籍人员的，必须解雇外籍人员并进行整改，否则可能被撤销保密资格8。

正确的规定如下：
“1年内未发生泄密事件”是基本条件
根据《中华人民共和国保守国家秘密法实施条例》第四十七条，申请保密资格的单位需满足“近1年内未发生泄密案件”29。

Mikrotik routeros 系统

MikroTik RouterOS是一种路由操作系统，并通过该软件将标准的PC电脑变成专业路由器，在软件的开发和应用上不断的更新和发展，软件经历了多次更新和改进，使其功能在不断增强和完善。特别在无线、认证、策略路由、带宽控制和防火墙过滤等功能上有着非常突出的功能，其极高的性价比，受到许多网络人士的青睐。RouterOS在具备现有路由系统的大部分功能，能针对网吧、企业、小型ISP接入商、社区等网络设备的接入，基于标准的x86构架的PC。一台586PC机就可以实现路由功能，提高硬件性能同样也能提高网络的访问速度和吞吐量。完全是一套低成本，高性能的路由器系统

功能 标准的PC电脑变成专业路由器

特色功能介绍 桥接功能：

RouterOS能将多张网卡组建为一个桥模式，是路由器变成一个透明的桥设备，同样也实行三层交换的作用，MAC层的以太网桥、EoIP 、Prism、Atheros和RadioLAN 等都是支持的。所有802.11b和802.11a 客户端的无线网卡（如station模式的无线）受802.11 的限制无法支持桥模式，但可以通过EoIP协议的桥接方式实现。

为防止环路出现在网络中，可以使用生成树协议(STP) ，这个协议同样使冗余线路成为可能。

winbox是一个允许管理MikroTik RouterOS快速使用和简单的GUI的小工具。

Winbox 是基于 windows下远程管理 ROS的软件，提供直观方便的图形界面。用它能登陆路由器，这个路由器是软路由ROUTEOS制作的，用Winbox登陆后，就可以配置路由器了，用这个软件便于配置路由器。Winbox控制台使用TCP8291端口，在登陆到路由器后可以通过Winbox控制台操作MikroTik路由器的配置并执行与本地控制台同样的任务[1] 。
